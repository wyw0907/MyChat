聊天客户端命令
login 登录，会提示输入用户名密码
logout 注销登录
regist 注册用户
show list 显示在线好友列表
show group 显示已经加入的讨论组
group list 当前讨论组在线好友列表
join [group] 加入讨论组
leave [group] 退出讨论组
create [group] 创建讨论组
delete [group] 删除讨论组
chatgroup [group] 进入讨论组聊天
chat [friend] 选择好友聊天
send file 发送文件，会提示输入发送给那个好友和要发送的文件名
show record 显示聊天记录
clean record 删除自己发 的聊天记录

ss管理端命令
INFO 日志输出级别为info
DEBUG 日志输出级别为debug
WANRING 日志输出级别为WANRING
ERR 日志输出级别为error
ReqList 显示在线用户列表
KILL [user] 踢用户下线
